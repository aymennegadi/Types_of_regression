# Types of Regression

This project compares five regression techniques:

- Linear Regression
- Polynomial Regression
- Ridge Regression
- Lasso Regression
- Elastic Net Regression

## Read the PDF First

Before running the examples, please read the PDF report included in this repository.

The report provides:
- Theoretical explanations
- Mathematical formulas
- MSE calculation
- Model comparison
- Interpretation of the graphs

## Contents

Each folder contains:
- Python code
- Example dataset
- Visualization of results

## Goal

The purpose of this project is to understand how different regression models work and compare their behavior on practical examples.
